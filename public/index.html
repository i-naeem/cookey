Cookey

A rest api to fetch Javascript generated cookies from websites. These cookies can then be used to scrape those website with simple fetch. 

            {
                "name": "Cookey",
                "description": "A rest api to fetch Javascript generated cookies from websites.",
                "router": {
                    "/api": {
                        "method": "GET",
                        "args": {
                            "url": {
                                "type": "string",
                                "required": true,
                                "description": "Website url you want to get cookies for.",
                            }
                        }
                    }
                }
            }
